<h2><a href=https://www.codewars.com/kata/5635e7cb49adc7b54500001c/train/javascript target="_blank">ATM</a></h2><h3>7 kyu</h3><p>An ATM has banknotes of nominal values 10, 20, 50, 100, 200 and 500 dollars.You can consider that there is a large enough supply of each of these banknotes.</p><p>You have to write the ATM's function that determines the minimal number of banknotes needed to honor a withdrawal of <code>n</code> dollars, with <code>1 &lt;= n &lt;= 1500</code>.</p><p>Return that number, or <code>-1</code> if it is impossible.</p><p>Good Luck!!!</p>

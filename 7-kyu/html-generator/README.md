<h2><a href=https://www.codewars.com/kata/54eecc187f9142cc4600119e/train/javascript target="_blank">HTML Generator</a></h2><h3>7 kyu</h3><p>Another rewarding day in the fast-paced world of WebDev. Man, you love your job! But as with any job, somtimes things can get a little tedious. Part of the website you're working on has a very repetitive structure, and writing all the HTML by hand is a bore. Time to automate! You want to write some functions that will generate the HTML for you. </p><p>To organize your code, make of all your functions methods of a class called HTMLGen. Tag functions should be named after the tag of the element they create. Each function will take one argument, a string, which is the inner HTML of the element to be created. The functions will return the string for the appropriate HTML element.</p><p>For example, </p><p>JavaScript:</p><pre><code class="language-javascript"><span class="cm-keyword">var</span> <span class="cm-def">g</span> <span class="cm-operator">=</span> <span class="cm-keyword">new</span> <span class="cm-variable">HTMLGen</span>();<span class="cm-keyword">var</span> <span class="cm-def">paragraph</span> <span class="cm-operator">=</span> <span class="cm-variable">g</span>.<span class="cm-property">p</span>(<span class="cm-string">'Hello, World!'</span>);<span class="cm-keyword">var</span> <span class="cm-def">block</span> <span class="cm-operator">=</span> <span class="cm-variable">g</span>.<span class="cm-property">div</span>(<span class="cm-variable">paragraph</span>);<span class="cm-comment">// The following are now true</span><span class="cm-variable">paragraph</span> <span class="cm-operator">===</span> <span class="cm-string">'&lt;p&gt;Hello, World!&lt;/p&gt;'</span><span class="cm-variable">block</span> <span class="cm-operator">===</span> <span class="cm-string">'&lt;div&gt;&lt;p&gt;Hello, World!&lt;/p&gt;&lt;/div&gt;'</span></code></pre><p>Your HTMLGen class should have methods to create the following elements:</p><ul><li>a</li><li>b</li><li>p</li><li>body</li><li>div</li><li>span</li><li>title</li><li>comment</li></ul><p>Note: The comment method should wrap its argument with an HTML comment. It is the only method whose name does not match an HTML tag. So, <code>g.comment('i am a comment')</code> must produce <code>&lt;!--i am a comment--&gt;</code>.</p>
